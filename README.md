# SIGN Impact Dashboard

Dieses Projekt ist das **SIGN Impact Dashboard**, ein interaktives Tool zur Berechnung von Zeit-, Kosten- und CO₂-Einsparungen durch digitale Signaturen.

## Features

- Calculator für Einsparungen
- KPI Cards & Animierte Charts
- ROI Simulator
- Benchmark Vergleich
- PDF Export
- Share Link Generator
- Landing Page & Admin Mode
- Vollständig responsive, Tailwind CSS
